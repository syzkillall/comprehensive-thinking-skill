# 质量检查清单

这个清单用于判断一次全面思考是否真正提升了复杂问题的结论质量，而不是把模板填满。

## 通过标准

一次合格的全面思考，至少应该满足以下条件：

- 重新定义了问题，而不是直接回答表层问题。
- 明确了本轮讨论对象、边界、成功标准和错误代价。
- 选择的理论体系真实参与判断，改变了问题定义、证据选择、方案排序或验证动作。
- 识别了关键矛盾、矛盾主次和判断主轴。
- 收集或指出了能改变判断的关键真实信息，而不是堆砌背景资料。
- 形成了当前问题自己的综合理论体系，而不是摘要外部资料。
- 给出了真实有力的最强反方意见。
- 对关键前提进行了来源标注和辩证分析。
- 说明了结论的适用范围、未知、风险和可推翻条件。
- 最终落到证据、实验、实现、测试、验收或文档沉淀之一。

## 常见失败形态

以下情况应视为没有完成全面思考：

- 只把五重审视当成栏目标题，内容没有真正改变判断。
- 只列理论名称，没有说明理论如何参与推理。
- 只归纳资料，没有演绎推理、竞争解释或前提分析。
- 反方意见很弱，无法挑战当前结论。
- 前提分析停留在“可能有风险”，没有导致降级、补证、边界收缩或重建判断。
- 最终输出只是观点、建议或任务清单，没有形成系统化理解。
- 结论没有说明如何验证，也没有说明什么事实会推翻它。

## 简单评分

可按 0-2 分给每项评分：

```text
0 = 缺失或形式化出现
1 = 出现但没有明显改变判断
2 = 明确改变判断、边界、证据或方案
```

评分项：

```text
问题重定义
领域与理论体系选择
关键事实收集
关键矛盾与判断主轴
综合理论体系
竞争解释或方案
最强反方意见
前提辩证分析
边界、风险与未知
可验证收束
```

建议判定：

```text
16-20 分：可认为是一次有效的全面思考
10-15 分：有结构，但需要补关键事实、反方或前提分析
0-9 分：大概率只是形式化套模板
```

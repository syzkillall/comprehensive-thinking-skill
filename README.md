## 为什么会有全面思考
AI 可以帮助我们解决问题，
但我们信任它的答案吗？
当我们遇到一些复杂问题时，AI 的表现有时候不尽如人意。我们可以看到 AI 在互联网上各处搜集信息，然后将其总结后提供给我们，但是这些答案却缺少信任感。这种信任感的缺失在两个层面：
1. 答案组织过程的粗糙-简短的推理后，给出结论摘要-这给人一种答案是由直觉带来的感受，并未进行仔细推敲
2. 结论的可靠性缺失:结论往往归纳多、演绎少，发散多、探究少，结论降级成了结束词。

而在一些场景中，老百姓的寻医问药、学者们对专业领域的研究、工程师们设计精密的系统，在这些场景之中，严肃的推理是必要的、粗糙的代价很大。
提升 AI 结论产出结论的质量是全面思考面对的课题。
